First Glance

Hello! My name is Maxine, and I'm currently attending the Girls Who Code Summer Immersion Program (SIP). I was introduced to the program through my computer programming teacher at school.

At first, I didn't like coding. I thought it tedious and redundant. I'm glad to be proven wrong - HTML was the first language I learned, and I still don't like it, but languages like C++ and Python are more appealing to me.
